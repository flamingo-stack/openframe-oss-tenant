OpenFrame Authorization Server

Welcome to OpenFrame Authorization Server

This is the central OAuth2/OIDC authorization server for the OpenFrame ecosystem. It provides secure authentication and authorization services for all OpenFrame applications.

Features:

  • ✅ OAuth 2.1 Authorization Server
  • ✅ OpenID Connect 1.0 Support
  • ✅ Local User Authentication
  • ✅ Federated Identity (Google OAuth2)
  • ✅ Role-Based Access Control (RBAC)
  • ✅ Multi-Tenant Ready Architecture
  • ✅ JWT Token with Custom Claims
Sign In

OAuth2/OIDC Endpoints:

  • Authorization: /oauth2/authorize
  • Token: /oauth2/token
  • JWK Set: /oauth2/jwks
  • User Info: /userinfo
  • Token Revocation: /oauth2/revoke
  • Token Introspection: /oauth2/introspect
  • OpenID Configuration: /.well-known/openid-configuration
